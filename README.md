# Argentum Intelligence Landing Page

A modern React landing page for a fictional AI company specializing in autonomous agents.

## Features

- Hero section with company introduction
- Interactive timeline and company metrics
- Revenue growth charts with Recharts
- Investor relations chat interface with mock AI responses
- Responsive design with dark theme

## Technologies

- React 19.2.4 with TypeScript
- Vite for development and building
- Recharts for data visualization
- Lucide React for icons

## Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173`

**Note:** This uses mock data and simulated AI responses. To integrate real services, update `services/geminiService.ts` with your API configuration.
